## 4.Home接口

#### 接口地址：./Home/Login
#### 调用方式：Get/Post
##### 调用方式一：
###### 请求参数：
参数名称      |参数说明      |参数类型      |
:---------: | :---------: | :---------: |
Email  	    | 用户Email    | String      |
Password    | 用户密码      | String      |
###### 返回参数：
参数名称   | 参数说明   | 参数类型    |
:-------: | :-------: | :-------: | 
Result    | 是否登录成功| bool      |

##### 调用方式二：
###### 请求参数：
参数名称      |参数说明      |参数类型      |
:---------: | :---------: | :---------: |
Name  	    | 用户名       | String      |
Password    | 用户密码      | String      |
###### 返回参数：
参数名称   | 参数说明   | 参数类型    |
:-------: | :-------: | :-------: | 
Result    | 是否登录成功| bool      |

>### 接口名称：获取Root目录下的所有文件和文件夹
#### 接口地址：./Home/Get

#### 调用方式：Get/Post
###### 请求参数：
参数名称      |参数说明        |参数类型        |
:---------: | :-----------: | :-----------: |
###### 返回参数：
参数名称   | 参数说明         | 参数类型   |
:-------: | :------------: | :-------: | 
DataList  | 文件夹以及所属文件| object    |
``` C#
DataList
{
	List<Folder> FolderList;
	List<File> FileList;
}
```

>### 接口名称：添加文件
#### 接口地址：./Home/Upload

#### 调用方式：Post
#### 请求头：enctype="multipart/form-data"
###### 请求参数：
参数名称     | 参数说明      |参数类型      |
:---------: | :---------: | :---------: |
FileList    | 文件结合     | List        |
``` C#
FileList:
	type: List<File>
	描述: 上传文件列表
File:
	type: System.Web.HttpPostedFileWrapper
	描述: 上传文件
```
###### 返回参数：
参数名称   | 参数说明   | 参数类型    |
:-------: | :-------: | :-------: | 
ResultList| 结果集	  | List      |
``` C#
ResultList:
	type: {
		Result: bool,
		Flie: File
	}
	描述:Result表示当前文件是否长传成功,
		File表示当前上传的文件
```
