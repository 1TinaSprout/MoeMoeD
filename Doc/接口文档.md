## User接口

>### 1. 接口名称：获取User
#### 接口地址：./User/Get

#### 调用方式：Get/Post
##### 调用方式一：
###### 请求参数：
参数名称      |参数说明        |参数类型        |
:-------: | :-----------: | :-------: |
Email		  |   用户Email   |   String   |
###### 返回参数：
参数名称   | 参数说明   | 参数类型   |
:-------: | :-------: | :-------: | 
Id        | 用户Id	  | int      |
Email     | 用户Email  | String  |
Name      | 用户名	  | String  |

##### 调用方式二：
###### 请求参数：
参数名称      |参数说明        |参数类型   |
:-------: | :-----------: | :-------: |
Name         |   用户名   |   String   |
###### 返回参数：
参数名称   | 参数说明   | 参数类型   |
:-------: | :-------: | :-------: | 
Id        | 用户Id	  | int      |
Email     | 用户Email  | String  |
Name      | 用户名	  | String  |

>### 1. 接口名称：获取File
#### 接口地址：./File/Get

#### 调用方式：Get/Post
##### 调用方式一：
###### 请求参数：
参数名称      |参数说明      |参数类型      |
:---------: | :---------: | :---------: |
Id  	    | 文件Id       | int         |
###### 返回参数：
参数名称   | 参数说明   | 参数类型    |
:-------: | :-------: | :-------: | 
Id        | 文件Id	  | int       |
UpdateTime| 文件更新时间| String    |
Name      | 文件名	  | String    |
Size      | 文件大小   | String    |
Type      | 文件类型   | String    |

##### 调用方式二：
###### 请求参数：
参数名称      |参数说明        |参数类型    |
:----------: | :-----------: | :------: |
UserId       | 用户Id        | int       |
###### 返回参数：返回此用户Root目录下的文件
参数名称   | 参数说明   | 参数类型   |
:-------: | :-------: | :-------: | 
FileList  | 文件列表	  | List      |

##### 调用方式三：
###### 请求参数：
参数名称      |参数说明        |参数类型    |
:----------: | :-----------: | :------: |

###### 返回参数： 返回此文件夹Id下的文件
参数名称   | 参数说明   | 参数类型   |
:-------: | :-------: | :-------: | 
FileList  | 文件列表	  | List      |
